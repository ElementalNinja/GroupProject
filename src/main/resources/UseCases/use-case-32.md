# USE CASE 32: A country report requires the following columns:    Code.  Name.  Continent.  Region.  Population.  Capital.

## CHARACTERISTIC INFORMATION


### Goal in Context
To allow a report on the following information about countries: code, name, continent, region, capital, population

### Scope

Company.

### Level

Primary Task

### Preconditions
1. All columns mentioned must be stored in the database

### Success End Condition
Report is generated and displayed successfully

### Failed End Condition
Report could not be displayed

### Primary Actor
User/Employee

### Trigger
User wishes to see a report with the mentioned information

## MAIN SUCCESS SCENARIO
Report displayed successfully

## EXTENSIONS
1. Report could not be displayed
    i. Check database has all required information
    ii. Revise query

## SUB-VARIATIONS

None.
