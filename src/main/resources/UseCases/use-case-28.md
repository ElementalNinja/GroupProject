# USE CASE 28: The population of a country.

## CHARACTERISTIC INFORMATION


### Goal in Context
To allow employees/users to find the population of a certain country
### Scope

Company.

### Level

Primary Task

### Preconditions
1. Specific country must be in database
2. Population for specific country must be in database

### Success End Condition
The population of specified country is displayed

### Failed End Condition
1. Population couldnt be displayed
2. Population number is inaccurate

### Primary Actor
User/Employee

### Trigger
User wishes to find out the population of a country

## MAIN SUCCESS SCENARIO
1. Country population is displayed accurately

## EXTENSIONS
