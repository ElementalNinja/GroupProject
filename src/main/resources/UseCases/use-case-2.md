# USE CASE 2: Display all the countries in a continent organised by largest population to smallest.

## CHARACTERISTIC INFORMATION


### Goal in Context
To allow employees to retrieve and look at a list of all countries in a continent, organised by largest population to smallest

### Scope

Company.

### Level

Primary Task

### Preconditions

Database contains information regarding all countries within the specific continent as well as having their population

### Success End Condition

List of the countries in the continent are displayed in the correct order to the employee

### Failed End Condition

List was unable to be displayed
List was displayed in the wrong order

### Primary Actor

Organisation Employee

### Trigger

The employee was told to gather population data for each country within a specific continent

## MAIN SUCCESS SCENARIO

1. Organisation instructed employee to get population information
2. Employee gets contintent population data is needed for
3. Employee retrieves population for each country in the continent

## EXTENSIONS

1. Countries are missing from database
   i. More countries must be added to database
2. Populations are missing from database
   i. Populations must be kept updated and new ones must be added

## SUB-VARIATIONS

None.
