# USE CASE 5: The top N populated countries in a continent where N is provided by the user. 

## CHARACTERISTIC INFORMATION

### Goal in Context
To allow users to enter a number then display the top ___ most populated countries within a specific continent

### Scope

Company.

### Level

Primary Task

### Preconditions

Database contains all country names, their continent and their population

### Success End Condition
List of the top countries in a specific continent where the number of countries is the amount specified by the user


### Failed End Condition
Wrong amount of countries displayed
Countries in wrong continent displayed
Countries displayed in wrong order


### Primary Actor

User

### Trigger

User wishes to see a list of the top populated countries in a specific continent

## MAIN SUCCESS SCENARIO
Full list is displayed successfully in correct order, amount and continent


## EXTENSIONS
1. Wrong amount of countries displayed. 2. Wrong continent displayed. 3. Countries displayed in wrong order
    i. Evaluate query again


## SUB-VARIATIONS

None.
